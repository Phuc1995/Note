generate ssh-keygen
1. Run the command: ssh-keygen -t rsa
  When it prompts for the pass word phase, press enter (Do not enter
any characters), public and private keys will be generated in ~/.ssh folder
with the below file names: id_rsa.pub, id_rsa 
2. Run the command: 

cd .ssh
cat id_rsa.pub >> authorized_keys

cd ../
chmod go-w ~/
chmod 600 ~/.ssh/authorized_keys
chmod 700 ~/.ssh

4. export these env: (chạy lại file bash là được)

SSH_USER_NAME=<user>

SSH_KEY_PATH=~/.ssh/id_rsa

AMSSFTPDB_USER_HOME_DIR=<user home_dir>

AMSSFTPDB_USER=<user>
